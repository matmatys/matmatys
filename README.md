# Architecture Demo Projects

## Wallet + Payment Processor System

A demo platform that simulates a **digital wallet integrated with a payment processor gateway**.

This project demonstrates **distributed system architecture** and integration patterns used in fintech platforms.

Architecture:

Frontend  
Angular

Backend services  
Java Spring Boot wallet service  
NestJS payment processor gateway

Infrastructure  
MySQL database  
Docker containerization

The gateway abstracts external processors and supports:

- simulated processor
- Stripe integration

Repository:

https://gitlab.com/matymatute_code/payment_processor_example_v2
